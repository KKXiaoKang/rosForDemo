### 关于线性代数的一些几何定律建议先看一下3blue1brown的B站油管主视频，用于几何化的理解线性代数的意义：用于空间变换


```
零矩阵
[0 0 0]
[0 0 0]  = O
[0 0 0]

零矩阵和普通矩阵A符合以下加法定则
A - O = A
A + O = A

单位矩阵
[1 0 0]
[0 1 0]  =  I
[0 0 1]

零矩阵和普通矩阵A、普通矩阵B 符合以下乘法定则
A I = A
I B = B
```
